Inputs

Distances are measured from the nut edge. Compensation values are scale-adjusted from your original model.

How It Works

Fret locations use equal temperament: each semitone divides string length by 2^(1/12). Bridge setbacks are scaled from a 650 mm reference: classical uses a straight saddle setback, while steel-string uses separate first/sixth values.
